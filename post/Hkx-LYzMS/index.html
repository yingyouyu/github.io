1.分析

题面不用写了吧
经典大水题
直接看代码

#include<iostream>
using namespace std;
int a[11],n,ans;
const int x=30;
int main(){
    cin>>n;
    n+=x;
    for(int i=1;i<=10;i++){
        cin>>a[i];
        if(a[i]<=n){
            ans++;
        }
    }
    cout<<ans;
    return 0;
}

校门外的树题解参见这里

靠的就是小学的植树问题，当然还有初中的数轴（doge，毕竟题目里有说）

所以说这题就是考加法和大小比较
