RQNOJ 14 校门外的树 题解

2020-10-24

1.题面

题目描述

某校大门外长度为L的马路上有一排树，每两棵相邻的树之间的间隔都是1米。我们可以把马路看成一个数轴，马路的一端在数轴0的位置，另一端在L的位置；数轴上的每个整数点，即0，1，2，……，L，都种有一棵树。
由于马路上有一些区域要用来建地铁。这些区域用它们在数轴上的起始点和终止点表示。　　已知任一区域的起始点和终止点的坐标都是整数，区域之间可能有重合的部分。现在要把这些区域中的树（包括区域端点处的两棵树）移走。你的任务是计算将这些树都移走后，马路上还有多少棵树。

输入格式

输入的第一行有两个整数：L（1 <= L <= 10000）和 M（1 <= M <= 100），L代表马路的长度，M代表区域的数目，L和M之间用一个空格隔开。接下来的M行每行包含两个不同的整数，用一个空格隔开，表示一个区域的起始点和终止点的坐标。

输出格式

输出包括一行，这一行只包含一个整数，表示马路上剩余的树的数目。

样例输入

500 3
150 300
100 200
470 471

样例输出

298

2.分析

本题是一道大水题，05年pj第一题（应该，毕竟这么水）
靠的就是小学的植树问题，当然还有初中的数轴（doge，毕竟题目里有说）
如果把本题的马路看成数轴的话，0坐标就是原点，正方向是已经确定的
然后计算共计多少棵树（其实这里作者吃瘪一次）
然后划分区间，把这些树移走，如果已经移走，就不做处理（难道你tm要挖到软流层？）
所以这题还是比较水的（至于陶陶摘苹果水还是这一题水，下次陶陶摘苹果我发题解时我会进行比较）

3.代码

这是自己写的

#include<iostream>
#include<cstring>
using namespace std;
int a[10086],b[10086],x,y;
bool vis[10086];
int ans;
int main(){
    memset(vis,true,sizeof(vis));//先确定树都没被移走
	cin>>x>>y;
	ans=x+1;//算一算有几棵树，其实这里就是小学植树问题
	for(int i=1;i<=y;i++){
		cin>>a[i]>>b[i];//输入要处理的区间
		for(int j=a[i];j<=b[i];j++){
			if(vis[j]==true){//如果树没被移走
				vis[j]=false;//移走
				ans--;//树数量减一
			}
		}
	}
	cout<<ans;//输出最后树的数量
	return 0;
}
